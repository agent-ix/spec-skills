# Verification & Evidence Strategy

This skill ensures that every requirement has a defined method of verification and a tangible output.

## Process

1.  **Verification Method Definition**:
    -   For each FR/SR/StR, define the verification method:
        -   Test
        -   Analysis
        -   Inspection
        -   Demonstration

2.  **Evidence Artifact Definition**:
    -   For each FR/SR/StR, define the evidence artifact:
        -   Test case
        -   Log
        -   Metric
        -   Report

3.  **Rule**:
    -   No implementation task may exist without an associated evidence artifact.
